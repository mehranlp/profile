Projects

Maximum Drawdown Dashboard
Maximum drawdown shows how much an investment drops from its peak before recovering. It’s a key metric for understanding potential losses and how long recovery might take. This project visualizes drawdown and wealth growth over time for assets like major US equity indexes, Global Equity, Gold, Global Fixed Income index, and Bitcoin, using steady annual contributions. It’s especially helpful for comparing investment strategies, assessing risk tolerance, and evaluating long-term consistency across funds.
Multi-asset portfolio constriction, backtesting and correlation heatmap
This interactive Streamlit app allows users to construct and analyze custom multi-asset portfolios using popular ETFs. Users can select assets, adjust their portfolio weights with intuitive sliders, and visualize the resulting expected returns, volatility, and efficient frontier. The app also includes detailed visualizations such as daily return distributions, a correlation heatmap, and a backtest comparison against the SPY benchmark. Historical data is sourced from Yahoo Finance via the yfinance API, and key performance metrics such as realized return and risk are computed to aid investment decision-making. This tool demonstrates practical applications of financial data analysis, portfolio optimization, and interactive visualization in Python.
